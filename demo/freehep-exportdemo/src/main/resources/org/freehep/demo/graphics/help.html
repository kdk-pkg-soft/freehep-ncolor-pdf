FreeHEP Graphics Export Demo

The FreeHEP Java library contains routines for outputting graphics in a variety of formats including:

  • PDF
  • Postscript and Encapsulated postscript
  • Scalable Vector Graphics (SVG)
  • Enhanced Meta-File (EMF)
  • MacroMedia Flash (SWF)
  • Bitmapped formats including GIF, PNG and JPG

The emphasis is on producing high quality vector graphics for use with scientific visualization applications, but the library should be usable in many other fields.

This demo application provides a number of different test displays which can be saved to a variety of different formats using the File, Export menu item. Many of the graphics formats provide a variety of options, which can be accessed using the Options... button on the export dialog, after selecting an output format.

On Windows the EMF format can be used to directly paste vector graphics into other windows applications, such as PowerPoint and Word. Use the Edit, Copy menu item to copy a page to the system clipboard, then paste it into your favorite application.
