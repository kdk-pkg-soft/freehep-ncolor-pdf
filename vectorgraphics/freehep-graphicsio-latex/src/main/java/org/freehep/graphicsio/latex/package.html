LaTeX Output Format.

Allows writing of single page LaTeX (PSTricks) graphics files using

  • LatexGraphics2D

The following Limitations exist:

  • Implementation of transparency is missing (probably cannot be supported).
  • TestTransforms: Ninth transform is translated and flipped.
  • Image inclusion not implemented.
  • Limited number of fonts supported.
  • Some text appears in the wrong position.
  • Text is not resized with the rest of the graphic.
  • PSTricks does not support line caps (with lineto) or line join styles.
  • PSTricks does not support repeating gradients or pattern fills.

@Status Stable.
