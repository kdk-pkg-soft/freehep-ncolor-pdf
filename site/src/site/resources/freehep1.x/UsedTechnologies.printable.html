FreeHEP Java Library

Components and Tools for Java

Used Technologies

The following set of external technologies are used in the FreeHEP Java Library (in alphabetical order):

  • ant - Make for Java from ant.pache.org
  • bcel - Byte Code Engineering Library from jakarta.apache.org/bcel
  • javacc - Java Compiler Compiler from www.webgain.com
  • java3d - Java3D extension from www.javasoft.com
  • junit - Unit testing framework from www.junit.org
  • servlet - Servlet library from www.javasoft.com
