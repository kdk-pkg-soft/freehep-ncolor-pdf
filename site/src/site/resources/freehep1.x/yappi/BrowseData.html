Browse Particle Data

The Java Servlet provides online information about the particle properties. Quick particle searches can be performed by specifying particle name using the encoding below or by PDGID number.

Particle ASCII Particle Encoding
name charge subset (special) name charge (subset) (special)
e+ e+
p+ pi+
h eta
hc(1S) eta(c)(1S)

You may browse particle data by name, e.g by typing "eta(c)(1S)", "eta", "pi+", and so on, in the name box below.
