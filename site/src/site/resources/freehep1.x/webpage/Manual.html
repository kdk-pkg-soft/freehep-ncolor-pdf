Quick Guide to the WebPage system

Installation

  • Get source code from FreeHEP CVS
  • add FREEHEP/tools to your PATH
  • type: webpage *.xhtml to run the tool

Configuration of a New Web

Copy the following files/directory from FREEHEP/web/freehep to your web directory:

  • webpage/image directory 
  • webpage/Menu.xml
  • webpage/Header.ivm
  • webpage/WebPage.css
  • webpage/PrintablePage.css

Your Own Pages

Write your content pages

Create your content web pages with the extension .xhtml in pure XHTML. You may add Velocity statements inside these pages.

For more information on the capabilities of Velocity see their User Manual.

Header.ivm

Edit the Header.ivm to create a header for printable and web documents, see example below:

#set( $webName = "WebPage" )
#set( $webSearch = "java.freehep.org" )
<h1>WebPage</h1>
Web Page Generation System

Offline generation of Web Pages

Type 

WebPage page1.xhtml page2.xhtml subdir/page3.xhtml

or

WebPage *.xhtml subdir/*.xhtml

which will convert your xhtml pages into page1.html, page1.printable.html, page2.html and printable/page2.html
