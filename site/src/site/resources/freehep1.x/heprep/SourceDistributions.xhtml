Source Code

For HEPREP C++ version 2.0.2: To (re-)build HEPREP C++ for a particular platform you need to get HEPREP-2.0.2-src.zip or .tar.gz file from the ftp://ftp.slac.stanford.edu/software/freehep/HEPREP/v2.0.2/ directory. Then follow the instructions in README-HEPREP.html contained in that distribution.

For HEPREP Java the source code is in the freehep repository.
