FreeHEP Java Library

Components and Tools for Java

Demonstrations

The following demos illustrate various features of the FreeHEP Java Library, although some are also useful applications in their own right. All require that Java Web Start 1.0 (or later) to be already installed on your computer.

  • IconBrowser - A demo of the FreeHEP application framework.
  • ExportDemo - A demo of the FreeHEP vector graphics export facility.
  • Root IO - Several demos are available
  • 3D Plotting package

More demos coming soon....

In addition there are some demos of the JAS plot widget (soon to migrate to FreeHEP and become the FreeHEP plot widget).
