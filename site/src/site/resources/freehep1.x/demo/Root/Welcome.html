FreeHEP ROOT IO Demonstrations

Demos

The following demos require that your have JavaWebStart 1.0 (or later) installed.

The Root Object Browser

The root object browser allows you to browse objects and data stored inside a Root file. It can read any Root file written with Root 3.00/6 or later, including files containing user-defined objects.

The Root Histogram Browser

The root histogram browser allows you to view histograms stored inside a Root file. It uses the JAS Plot widget to display the histograms.

Please report any bugs you find in these applications to tonyj@slac.stanford.edu.
