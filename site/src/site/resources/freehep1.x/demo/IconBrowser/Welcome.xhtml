FreeHEP Icon Browser

The FreeHEP Icon Browser was written as a test application for the FreeHEP application framework, but is hopefully also a useful tool for developers using the JFA Icon collection, the Java Look and Feel Graphics, or any other icon collection stored in zip or jar files.

Currently the IconBrowser is available for download using Java Web Start. You must have Java Web Start 1.0 already installed on your machine for the download to work.

Feel free to send comments/suggestions/bug reports to the author, Tony Johnson.
