Geant4 Distributions

For Geant4.6.x use JAIDA 3.2.x and AIDAJNI 3.2.x and follow the instructions in the release notes of AIDAJNI.

For Geant4.5.x use JAIDA 3.0.x which is distributed as an overlay product. The following Geant4-JAIDA distributions (.zip or tar.gz files) can be downloaded. Platform specific distributions do NOT contain setup files for other platforms.

  • Download Geant4-JAIDA-3.0.2 for Windows
  • Download Geant4-JAIDA-3.0.2 for Linux
  • Download Geant4-JAIDA-3.0.2 for MacOS X
  • Download Geant4-JAIDA-3.0.2 for Solaris
  • Download Geant4-JAIDA-3.0.2.zip for All Platforms
  • Download Geant4-JAIDA-3.0.2.tar.gz for All Platforms
