JAIDA 3.0.1 Tutorial

Creating an Analysis Factory

Before you can use any of the AIDA interfaces you must start by creating an Analysis Factory. You can do this in the standard way using the following line of code:

IAnalysisFactory factory = IAnalysisFactory.create();

There are actually three different analysis factories included with JAIDA 3.0.1, with varying degrees of GUI support

  1. hep.aida.ref.AnalysisFactory - (in freehep-hep.jar) It has no GUI support. This is suitable for use in completely non-graphical environments, such as batch processing. A dummy IPlotter implementation is provided to satisfy the AIDA interface requirements, but no plots will be displayed.
  2. jas.aida.JASAnalysisFactory - (in jas.jar) A minimal implementation of IPlotter is provided using the JAS plotter. Each IPlotter will appear in its own top level window. Minimal interaction with the plots is allowed through popup menus.
  3. jas.aida.gui.JASGUIAnalysisFactory - (in JASAIDA.jar) A slightly more elaborate GUI is provided, in which each IPlotter appears within a top level window, and the complete tree of AIDA objects is displayed.
