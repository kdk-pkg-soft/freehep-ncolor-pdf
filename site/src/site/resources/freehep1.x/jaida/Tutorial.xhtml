Tutorial

In this tutorial we show how to run JAIDA from Java, C++, Jython using the same code that creates and fills two IHistograms and two IClouds. The IHistogram1D is also fitted using two different engines: Uncmin and Minuit (to use Minuit you need to download platform specific CERNLIBAdapter).

A special example on how to run JAIDA within Geant4 can be found under Geant4 and JAIDA.

We assume that the appropriate JAIDA version has been downloaded and that the CLASSPATH has been set to include the appropiate jar files as explained in the release notes.
