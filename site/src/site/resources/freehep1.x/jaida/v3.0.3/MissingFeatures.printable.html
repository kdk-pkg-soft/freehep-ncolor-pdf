AIDA in Java

Missing Features

The following is a list of the AIDA methods/classes that are not currently supported in JAIDA.

  • Styles classes implemented only minimally
    • IAxisSyle and IBaseStyle are partially implemented (it is possible to set logaritmic or linear scale axis)
    • IBrushStyle, IDataStyle, IFillStyle, IInfoStyle, ILineStyle, IMarkerStyle, ITextStyle, ITitleStyle are not implemented.
  • Class IInfo is not implemented
  • IFitter supports only constraints of the type "a = b"
  • IFunctions: no operations among built in functions are supported e.g.: "g+p1"
  • IHistogramFactory: createCopy(String, ... ) methods are not implemented
  • IPlotter, IPlotterFactory, IPlotterStyle, IPlotterRegion have partial implementations due to the missing styles classes.
  • IPlotterLayout is not implemented
  • ITree: the following methods are not implemented
    • mv(...)
    • setOverwrite(...)
    • cp(...)
    • symlink(...)
  • IEvaluator and IFilter can only be applied to ITuple's columns at the root level (not to columns inside folders)
  • ITuple: for the reason above the project methods on ITuple apply only to root level columns
