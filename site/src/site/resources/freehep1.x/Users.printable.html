FreeHEP Java Library

Components and Tools for Java

Users

Current users of the FreeHEP Java Library include the following individuals and projects:

  • JAS - Java Analysis Studio, SLAC, Stanford, California, U.S.A.
  • WIRED - WWW Interactive Remote Event Display, SLAC, Stanford, California, U.S.A.
  • GraXML - Java3D graphics display tool using XML, Julius Hrivnac, LAL, Orsay, France.
  • Geant4 - Simulation Toolkit.
  • Magnetic Field project for the ATLAS Detector at CERN, Andrea Formica.
  • Matrex, A graphical tool to show, organize and calculate big amounts of data, Andrea Ferrandi.

If you are a user of the FreeHEP library and wish to make add a reference to this list, send an e-mail to developers@freehep.org including the name, author, a one line description and a URL to the projects web site.
