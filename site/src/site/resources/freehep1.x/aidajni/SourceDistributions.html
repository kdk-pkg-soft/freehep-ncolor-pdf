For version 3.2.4: To (re-)build AIDAJNI for a particular platform you need to get AIDAJNI-3.2.4-src.zip or .tar.gz file from the ftp://ftp.slac.stanford.edu/software/freehep/AIDAJNI/v3.2.4/ directory. Then follow the instructions in README-AIDAJNI.html contained in that distribution.

For older versions (before 3.0.4) the source code for AIDAJNI is in the freehep repository, so you can build your own libraries on the platform of your choice.

  • download the desired distribution of the freehep code
  • from the head of the distribution just type "gmake dist"
  • use the AIDAJNI-{version}-{platform} file that is created
