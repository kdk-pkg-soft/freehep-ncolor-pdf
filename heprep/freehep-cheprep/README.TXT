**********************************************************************
 C++ implementation of the HepRep (version 2) driver.

 Author: Mark Donszelmann
**********************************************************************

This implementation allows a C++ program to build HepRep trees in memory,
browse them and to write HepRep XML and Binary HepRep files.

For documentation on HepRep (version 1 and 2) see:
http://heprep.freehep.org/

An example is available under the name: HepRepExample.cc

The distribution is available for: Linux, Windows and MacOS X.

There exists also a pure-java version.

Sources are available from:
http://java.freehep.org/
